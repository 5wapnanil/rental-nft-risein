## 📌 Project Description

This project implements a basic **Rental NFT system** using Soroban smart contracts on the Stellar network.

The idea is to allow NFT owners to temporarily lend (rent) their NFTs to other users for a fixed duration without transferring ownership.

---

## ⚙️ What it does

- Mint NFTs with unique token IDs
- Allow owners to rent out NFTs
- Assign temporary users (renters)
- Automatically revert usage rights back to owner after expiry

---

## ✨ Features

- 🪙 NFT Minting  
- 🔁 Ownership Transfer  
- ⏳ Time-based Rental System  
- 👤 Dual Role System:
  - Owner (permanent)
  - Renter (temporary)
- 🔍 Dynamic User Resolution:
  - Returns renter if rental active
  - Returns owner if expired
